# FinTrack Pro 💰

A personal finance tracker app built with Flutter and Firebase.

## Features
- 🔐 Sign up with OTP email verification
- 📊 Dashboard with pie chart
- 💸 Add income and expenses
- ⚠️ Budget alerts
- 🔔 Reminders with sound
- 📈 Monthly report
- 🌙 Dark/Light theme toggle
- ☁️ Firebase cloud database

## Tech Stack
- Flutter & Dart
- Firebase Authentication
- Cloud Firestore
- EmailJS
- Provider
- FL Chart
